Ниже представлена инструкция для запуска девелопер версии проекта. 
Клиентские версии приложения для корректной работы требуют наличие запущенного в той же сети сервера.

Для запуска Django сервера требуется:
1. Установить Python версии 3.6 (разработка проводилась на версии 3.6.8)
2. Перейти в директорию сервера Application/waifu2rest/
2. Установить все необходимые пакеты Python (pip install -r requirements.txt в директории waifu2rest)
3. Выполнить миграцию:
py manage.py makemigrations
py manage.py migrate
4. Запустить сервер на локальном порте:
py manage.py runserver 0.0.0.0:8000

Админ-панель можно будет открыть по адресу 127.0.0.1:8000/admin
Логин администратора: root, пароль: root


Для запуска веб версии приложения требуется:
1. Установить Node.js
2. Перейти в директорию веб-приложения angular Application/waifu2angular
3. Установить ангуляр: npm install angular
4. Установить зависимости: npm install
5. Запустить тестовый сервер: ng serve 

Веб версия будет доступна по адресу 127.0.0.1:4200

Для запуска мобильной версии приложения требуется: 
1. Установить Node.js
2. Установить Android SDK
3. Установить graddle
4. Установить ангуляр: npm install angular
5. Установить Сordova
5. Перейти в директорию мобильного приложения Application/waifu2mobile
6. Установить зависимости: npm install
7. Подключить телефонное устройство к компьютеру
8. Включить режим разработчика на телефоне
9. Находясь в папке мобильного приложения  собрать приложение и запустить на телефоне:
cordova run --device
